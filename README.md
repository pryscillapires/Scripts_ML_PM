# Risk-Controlled ML for PM Stability Maps

This repository contains scripts associated with the manuscript:

**Risk-Controlled Machine Learning for Circumbinary Stability Maps in the Patroclus–Menoetius System**

## Summary

This project develops a risk-controlled machine-learning surrogate for finite-horizon circumbinary stability screening in the Patroclus–Menoetius system.

The workflow includes:
- heliocentric N-body truth integrations,
- feature extraction,
- leakage-safe cross-validation,
- probability calibration,
- audit-set evaluation,
- and figure/table generation.

The surrogate is intended as a **finite-horizon, epoch-specific, phase-slice screening tool**.
